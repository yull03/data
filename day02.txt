1. 주석(Comment)
(1) 소스 코드를 설명을 써 넣는 것
(2) 컴파일 시에 사용 안 함
(3) 주석을 만드는 목적
    -변수, 함수, 로직에 설명을 적기 위해 
    -프로그램 시작 부분에 개요 설명
    -프로그램 로직이 매우 복잡
    주석ㅇ으로 플로우(흐름)을 미리 기술
    조금씩 프로그램으로 완성
    (Pseudo Coding : 수도 코딩)
    -실행할 부분을 테스트 시 잠시 제외하는 용도
(4) 한 줄 주석, 구간 주석, 문서화 주석
    한 줄 주석: //
    구간주석 : /*      */
    문서화 주석 : /** */

2. 변수(Variable)
    - 변수 : 별할 수 있는 수 <-> 상수()Constant
    - 크게 문자형과 숫자형으로 나누어짐
        문자 1 + 1 = 11 숫자 1 + 1 = 2
    - 문자형 : char, String
    - 숫자형 : int, long, float, double.  5 = int / 3.11 = double 
             int : -21억xxxx ~ 21억xxxx 
             long : int로 표현 할 수 없는 숫자 범위
             float, double : 소수점이 있는 숫자형으로
    - 기타형 : boolean(true, false 값만 가짐)

    -변수 이름 명명 규칙
        숫자로 시작 X : num1(O), 1num(X)
        특수문자는 $와  _만 허용
        자바의 reserveed word(예약어) 불가(예:class)

        myAge : 첫 글자는 소문자로 시작
                단어가 바뀌면 첫 글자를 대문자로
                myOldCamara
                Camel 포기법이라고 함(낙타의 혹 같이 취어나와서 이름이 정해짐)

        my_age : 모든 글자를 소문자로 통일
                 단어들을 _로 연결
                 my_old_camera
                 Snake  표기법이라고 함수
